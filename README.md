# 🌍 Equinox – AI Travel Planner

Equinox is an AI-powered travel planner that creates personalized travel itineraries based on a user's destination, budget, travel dates, interests, accommodation preferences, transportation choice, and travel style.

The project uses the Google Gemini API to generate customized travel plans with a modern, responsive interface.

---

## 🚀 Live Demo

🔗 https://equinox-travelplanner.netlify.app

---

## ✨ Features

-  AI-powered itinerary generation using Google Gemini
-  Personalized day-wise travel planning
-  Budget-based recommendations
-  Accommodation suggestions
-  Transportation recommendations
-  Weather recommendations
-  Packing checklist
-  Fully responsive design
-  Interactive loading animation
-  Printable itinerary

## 🎯 How It Works

1. Enter travel details.
2. Select interests and preferences.
3. Gemini AI processes the request.
4. Personalized itinerary is generated.
5. View recommendations for:
   - Hotels
   - Transportation
   - Weather
   - Packing essentials
   - Day-wise schedule
